# 🚀 My Python Automation Toolset
> 个人办公自动化脚本集合，旨在用 Python 终结重复性劳动。

---

## 📂 核心工具介绍

### 1. 📧 MailIndex Auto-Archiver (邮件归档助手)
**文件名：** `一键打包邮件归档.zip`
- **功能简述**：一键扫描本地海量邮件文件夹，自动提取文件清单并生成带超链接的 Excel 索引表。
- **核心逻辑**：
  - 自动过滤干扰项（如 `__pycache__`）。
  - 利用 `openpyxl` 构建原生 Excel 超链接，即点即看。
  - Windows 系统原生弹窗交互，流程丝滑。

### 2. 🖱️ FlashClicker RPA Lite (闪电手 RPA)
**文件名：** `Absolute_Zero.py`
- **功能简述**：基于 Win32 API 的轻量级 RPA 执行引擎，模拟物理键鼠操作。
- **适用场景**：老旧企业系统、内网网页等无法通过接口获取数据的场景。

---

## 🛠️ 环境依赖
运行本项目脚本前，请确保已安装以下依赖：
- **Python 3.x**
- **openpyxl** (用于处理 Excel 报表)
- **ctypes** (内置模块，无需额外安装)
